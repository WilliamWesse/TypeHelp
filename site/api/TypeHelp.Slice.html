    Remarks

    Constructor parameter array boundary violations in conjunction with the StrictLength and StrictOffset options MAY raise a TypeInitializationException with an inner InvalidOperationException.

    Constructors

    Name Description
    Slice(byte[], int, int)

    Create a Slice using length, offset and the default options:

    • data is in Network Byte Order (NBO), a.k.a., big-endian.
    • offset is locked, data MAY be truncated.
    Slice(byte[], int, int, SliceOpts?)

    Create a Slice using length, offset and the selected SliceOpts.
    Slice(byte[], TypeCode, int, SliceOpts?)

    Create a Slice using the selected TypeCode typeCode, offset and selected SliceOpts.

    If the typeCode is Object, a BigInteger will be assumed, using the equivalent (BigInteger).
    Slice(byte[], TypeCoex, int, SliceOpts?)

    Create a Slice using the selected TypeCoex, offset and selected SliceOpts.

    Fields

    Name Description
    Defaults

    The SliceOpts None value (0).
    LockLittleEndian

    A bitwise OR of the SliceOpts LittleEndian, LockedLength and LockUnsigned flags.
    Locked

    A bitwise OR of the SliceOpts LockedLength and LockUnsigned flags.
    Strict

    A bitwise OR of the SliceOpts StrictLength and StrictOffset flags.

    Properties

    Name Description
    AllowSigned

    true if the LockUnsigned flag is not set, otherwise false.
    Data

    A copy of the extracted byte array in big-endian order.

    If the AllowSigned Property is false and the HighBit Property is true, an additional zero-value byte will be present in the high-order (first) byte of the returned big-endian array.
    HighBit

    The value is true if the first (big-endian normalized) data byte has the high-order bit set, otherwise false.
    IsEmpty

    The value is true if the Data Property length is zero, otherwise false.
    IsExact

    The value is true if Length or Offset were not modified during initialization, else false.
    IsLittleEndian

    The value is true if the LittleEndian flag is set, indicating the Data Property byte order is the inverse of that extracted from the data provided to the constructor, otherwise false.
    IsSigned

    The value istrue if the HighBit Property is true and the LockUnsigned flag is set, otherwise false.
    IsUnsigned

    The value istrue if the HighBit Property is true and the LockUnsigned flag is set, otherwise false.
    Length

    The length of the byte array.
    LengthDelta

    The change, if any, made to the Length during initialization.
    Offset

    The position of the data in the input array.
    OffsetDelta

    The change, if any, made to the Offset during initialization.
    Options

    The SliceOpts provided to the constructor.
    TypeCode

    The TypeCode of the extracted data. This will be one of the following:

    TypeCode.Boolean
TypeCode.Byte     TypeCode.Char     TypeCode.DateTime
TypeCode.DBNull   TypeCode.Decimal  TypeCode.Double
TypeCode.Int16    TypeCode.Int32    TypeCode.Int64
TypeCode.Object   TypeCode.SByte    TypeCode.Single
TypeCode.UInt16   TypeCode.UInt32   TypeCode.UInt64
    TypeCoex

    The TypeCoex of the extracted value. This will be one of the following:

    TypeCoex.BigInteger  TypeCoex.Boolean  TypeCoex.Byte
TypeCoex.Char        TypeCoex.Complex  TypeCoex.DateTime
TypeCoex.DBNull      TypeCoex.Decimal  TypeCoex.Double
TypeCoex.Int16       TypeCoex.Int32    TypeCoex.Int64
TypeCoex.Object      TypeCoex.SByte    TypeCoex.Single
TypeCoex.UInt16      TypeCoex.UInt32   TypeCoex.UInt64
    TypeOf

    The Type of the extracted value.
    Value

    The value will be one of the following types:

    System.Boolean   System.Byte    System.Char
